# Hotel Notification Management System - Backend

## Project Overview

This is the backend application for the **Hotel Notification Management System**, built using **Spring Boot**. It provides RESTful APIs to manage notifications and calculate performance metrics such as impressions, clicks, CTR (Click-through Rate), and CVR (Conversion Rate).

## Features

- **CRUD operations** for managing hotel notifications.
- **Performance Metrics** auto-generation for the past 3 months.
- **REST API** to fetch notification performance over a specific time period.

## Prerequisites

Make sure you have the following installed:

- **JDK**: v11 or higher
- **Maven**: v3.6.x or higher
